# reflex-actuator
A 3D-printable high-torque planetary actuator.

# Features
- Continuous Torque: 80 Nm
- Peak Torque: 120 Nm 
- Weight: 1.3 Kg
- Operating Voltage Range: 24 VDC - 52 VDC
- Diameter: 190 mm
- Height: 60 mm
- Encoder: AS5047P
